# Marketing Classifier Comparison

This project compares the performance of four supervised learning classifiers:

- K-Nearest Neighbors
- Logistic Regression
- Decision Trees
- Support Vector Machines

The dataset contains marketing interactions related to bank product outreach.  
The goal is to evaluate model performance and understand which classifier performs best under different conditions.

## Repository Structure
- `notebooks/` — Jupyter notebooks for analysis
- `src/` — optional Python modules for reusable code
- `data/` — dataset storage (not included in repo if private)

## Getting Started
Open `notebooks/classifier_comparison.ipynb` to begin.
